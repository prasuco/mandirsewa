# mandirsewa

A platform made from the groundup to support mandir mandirs in nepal to accept donation through their own website.

### Technology Stack:

- `php`: for server side logic.
- `tailwindcss`: for clean utility based styling
- `jQuery`: for easy reactivity(probably, if not; hail vanilla javascript).
- `jquery-modal`: for easy modal popups using jquery
- `@prasuco/easy-sewa`: for initiating esewa payments.
- `quill.js`: for rich text editor
- `fontawesome`: who doesnot like icons?
